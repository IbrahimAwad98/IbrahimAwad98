## 📌 Featured Projects

### 🚗 [Road Rumble](https://github.com/IbrahimAwad98/Road-Rumble) — C · SDL2 · UDP Networking
Multiplayer 2D top-down racing game with a separate server and client binary, supporting up to 4 simultaneous players over UDP. Features split-screen gameplay, collision detection, lap tracking, tile-based rendering, and cross-platform Makefiles (Linux / macOS / Windows). Built as a 6-person group project managed with SCRUM.

### 📚 [Library Management — MongoDB](https://github.com/IbrahimAwad98/LibraryMongoDB) — Java 21 · JavaFX · MongoDB
Desktop library system built on full MVC architecture. Supports user authentication, multi-criteria book search (title, ISBN, author, genre, rating), text reviews, 1–10 ratings with automatic average calculation, and async UI updates via JavaFX `Task`. Built at KTH as part of the Database Engineering course.

### 📚 [Library Management — MySQL](https://github.com/IbrahimAwad98/LibraryMySQL) — Java 21 · JavaFX · MySQL
Relational counterpart to LibraryMongoDB — demonstrates SQL schema design, JDBC integration, and MVC architecture alongside the NoSQL version for direct SQL vs NoSQL comparison.

### 🔢 [Algorithms & Data Structures](https://github.com/IbrahimAwad98/Algorithms-and-data-structure) — Java
From-scratch implementations of Stack (3 variants), Queue, Linked List (singly & doubly), Binary Search Tree, QuickSort, MergeSort, ShellSort, backtracking maze solver, BFS brick-puzzle solver, and a postfix/infix expression evaluator. No external libraries.

### 🔬 [RISC-V Embedded Programming](https://github.com/IbrahimAwad98/RISC-V-) — RISC-V Assembly · C
Five progressive lab assignments on the GD32VF103 (Longan Nano) microcontroller. Covers ALU operations, GPIO bit manipulation, DAC/PWM control, buffered USART serial communication, SPI LCD driving, ADC sampling, and ECLIC vectored interrupt handling.

### 🌤️ [Weather App](https://github.com/IbrahimAwad98/Weather-App-via-React) — Next.js 16 · React 19 · TypeScript · Tailwind CSS
Real-time weather dashboard using the App Router pattern, server-side API integration, and responsive styling with Tailwind CSS v4.
